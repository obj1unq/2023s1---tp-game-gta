## Reglas de Juego

### Instrucciones
1. Presione la tecla "S" para comenzar a jugar.
2. Presione la barra espaciadora para hacer que Crash salte los obstaculos.
3. Para aumentar su vida, debe recoger las cajas de vida que caen del cielo.
